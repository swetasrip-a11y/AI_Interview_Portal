## üîë Key Concepts

### Real-Time Multimodal Interview
A system that simultaneously:
- Captures speech with real-time transcription
- Analyzes facial data and detects emotions
- Processes text-based chat with AI responses
- Provides real-time engagement scoring
- All happening simultaneously during an interview

### Components
1. **Speech Service** - Transcription, fluency analysis
2. **Facial Service** - Face detection, emotion classification
3. **Chat Service** - AI responses, quality evaluation
4. **API Routes** - 6 REST endpoints for real-time processing
5. **Frontend Component** - Interactive interview interface
6. **Dashboard** - Real-time metrics display

### Scoring
- Speech: 35% (fluency, confidence, technical depth)
- Facial: 35% (eye contact, emotions, engagement)
- Chat: 30% (relevance, clarity, completeness)
- **Final: Weighted average (0-100)**
